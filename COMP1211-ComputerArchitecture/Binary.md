## Definitions
- Bit: binary digit
- Byte: 8 bits
- Word: 32 or 64 bits (most use 64)
- Range(R): number of represented numbers
- K: 2<sup>10</sup> or 1024
- Radix Point: like decimal point but for all number systems

| Bits | Range             |
| ---- | ----------------- |
| 1    | 2(1 and 0)        |
| 4    | 16(0 to 15)       |
| 8    | 256               |
| 10   | 1024(1K)          |
| 16   | 65536(64K)        |
| 20   | 1,048,576(1M)     |
| 32   | 4,294,967,296(4G) |
